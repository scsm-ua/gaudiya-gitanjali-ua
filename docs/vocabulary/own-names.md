# Словник: Власні назви та імена

Правила застосування словника: 
    - імена і назви в тексті можуть складатись з меншої кількості слів ніж у словнику
    - у тексті допускається інший порядок слів імені чи назви
    - обовʼязково має співпадати транслітерація літер
    - перед іменами може стояти додаткове "Шрі" або "Шріла"
    - деякі загальні слова пишуться з великої літери, як власне імʼя, якщо мається на уважі божественна особистість

Правила форматування:
    - повною назвою слід вважати найбільшу кількість слів підряд, що відносяться до однієї назви
    - власні назви і імена не виділяються ніяким форматуванням (без курсиву, без жирного, без лапок)
    - частина повної фрази імені може складатись із іншомовного терміну, тоді термін треба виділити по правилам форматування термінів, не розділяючи пробілом
