# Словник: Слова іноземного походження

Правила застосування словника: 
    - іноземні фрази можуть складатись з меншої кількості слів ніж у словнику
    - у тексті допускається інший порядок слів у фразі
    - обовʼязково має співпадати транслітерація літер

Правила форматування:
    - одне поняття чи фраза має виділятись *курсивом*
    - інші форми виділення не допускаються (без жирного, без лапок)
    - якщо зустрічається фраза, одне з слів якої імʼя, або українське слово, а інше - іноземне слово, то слід виділяти тільки іноземне слово. Наприклад: 
      - Крішна-*према*
      - Ґауда-*деша*
